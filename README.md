## Student Loan and Payment Management System
A simple web-based Student Loan and Payment Management System built with PHP, MySQL, HTML, CSS, and JavaScript. The system allows users to manage student records, create and monitor loans, record payments, and track outstanding balances through an easy-to-use interface.

## Features
Student Management (CRUD)
Student Loan Management
Loan Payment Recording
Outstanding Balance Tracking
MySQL Database Integration
Tech Stack
Frontend: HTML5, CSS, JavaScript
Backend: PHP
Database: MySQL

## Purpose
This project was developed as a learning project to demonstrate CRUD operations, relational database management, and basic web application development using PHP and MySQL.
